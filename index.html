15. What is formatting in HTML?

Formatting in HTML refers to the use of HTML tags to structure and style text content, making it more readable, organized, and visually appealing. Formatting tags help to emphasize, organize, or highlight certain sections of text, and they serve both functional and aesthetic purposes. Improves Readability, Adds Emphasis, and Accessibility, Enhances Aesthetics: Creates a visually appealing layout, contributing to a professional look and feel.

20. What is the label tag, and how is it associated with form inputs?

The label tag in HTML is used to define labels for form inputs, providing a user-friendly way to identify each field. Associating labels with form elements improves accessibility and makes forms easier to navigate, especially for screen readers.

The iframe tag in HTML is used to embed another HTML document within the current one. It’s commonly used to display content like maps, videos, and other web pages within a section of a webpage.
